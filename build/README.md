# Build Icons

Place your application icons in this directory:

- **icon.icns** - macOS icon (512x512 or larger)
- **icon.ico** - Windows icon (256x256 or larger)  
- **icon.png** - Linux icon (512x512 PNG)

You can generate icons from a single source image using tools like:
- https://www.electronforge.io/guides/create-and-add-icons
- https://www.img2go.com/convert-to-ico
- https://cloudconvert.com/png-to-icns

For now, the build will use default Electron icons if these files are missing.
